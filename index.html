MEMORY

Bienvenue dans le jeu du Memory !

Comment jouer:

Sélectionnez un nombre de cartes et appuyez sur le bouton pour commencer la partie. Cliquez sur une carte pour afficher sa valeur puis sur une autre carte. Si les deux valeurs sont les mêmes, alors ces cartes sont retirées du jeu. Si les deux valeurs sont différentes, les cartes redeviennent invisibles et le compteur d'erreurs est incrémenté. Ce compteur d'erreur consitute votre score final, donc plus il est proche de 0, mieux c'est ! Le jeu se termine lorsque toutes les paires de cartes ont été trouvées.
